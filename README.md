# AnyT-d3-Repository

**AnyT-d3-Repository** (or **AnyT-d3**) is a set of **D3.js**-related technical solution library. AnyT-d3 helps with design ideas and solutions based on SVG, Canvas and HTML in response to different scenarios of the development practice. AnyT-d3 will inspire innovation in visualization and interactive technology.
